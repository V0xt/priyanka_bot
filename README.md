# JS chat bot for Discord

A simple discord bot which helps you manage your server by providing useful commands.

## Available commands:
  
!help    
!help [someTopic]   
!multiply [2] [4] [..]     
!ping	  
!8ball  
!fortune  
!bitcoin  
!weather [cityName] (currently not working) 
!play [youtubeURL]
!stop
!skip
