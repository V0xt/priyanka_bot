# JS chat bot for Discord

A simple discord bot which helps you manage your server by providing useful commands.

## Available commands:
  
!help    
!info    
!weather cityName    
!ping  
!8ball   
!fortune  
!bitcoin  
!play youtubeURL plays music from youtube link
!stop  
!skip  
!nowplaying  
!userinfo  
!report [your message]
